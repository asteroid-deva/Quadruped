# Quadruped
This repository houses the code and development progress for a series of custom quadruped robots.
* Phase 1 (QUADRUPED mini): A compact, 8-servo robot utilizing an ESP32 microcontroller and a PCA9685 driver. It features 2 Degrees of Freedom (2-DOF) per leg powered by EMAX ES08MA II servos. The system includes custom open-loop kinematics for standing, sitting and a shuffle/creep walking gait. It also features closed-loop control using a BNO085 IMU for dynamic, active self-leveling based on Pitch and Roll.

* Phase 2 (VISION-ASSISTED QUADRUPED): A distributed AI-assisted quadruped robotics system utilizing a Raspberry Pi 4B, ESP32, Intel RealSense camera, BNO085 IMU and dual PCA9685 servo drivers. The Raspberry Pi performs real-time computer vision and gesture recognition using OpenCV and MediaPipe, while the ESP32 handles deterministic low-level motor control and IMU-assisted posture stabilization. Hand gestures are mapped to predefined robot behaviors such as standing, sitting and waving through serial communication between the Raspberry Pi and ESP32. The system also integrates smooth servo interpolation, dual I2C bus architecture, Flask-based telemetry streaming, GPS support using Neo-6M and experimental open-loop gait development for future autonomous locomotion research.
