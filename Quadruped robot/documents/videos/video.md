# 📺 Demo Highlights

## ✅ Gesture-Controlled Standing
The robot transitions into a stable standing pose using MediaPipe hand gesture detection.

## ✅ Gesture-Controlled Sitting
A closed resting pose triggered through computer vision commands.

## ✅ Wave Gesture
Single-leg lifting with compensation balancing from remaining legs.

## ✅ IMU-Assisted Stabilization
Real-time posture correction using BNO085 pitch and roll feedback.

## ✅ Distributed Robotics Architecture
High-level AI processing on Raspberry Pi with low-level deterministic control on ESP32.
